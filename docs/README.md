### 할인 이벤트 상세 내용

- [X] 중복 할인 가능
- [X] 크리스마스 디데이 할인 (12.1 ~ 12.25)
    - [X] 1일 1,000원 부터 시작하여 크리스마스까지 1일당 100원씩 할인 금액 증가
    - [X] 총 주문 금액에서 할인
-  [X] 12월 할인 (12.1 ~ 12.31)
    - [X] 평일 할인 (매주 일 ~ 목)
        - [X] 총 주문 금액에서 디저트 메뉴 1개당 2,023원 할인
    - [X] 주말 할인 (매주 금 ~ 토)
        - [X] 총 주문 금액에서 메인 메뉴 1개당 2,023원 할인
    - [X] 특별 할인 (⭐데이: 매주 일요일 및 크리스마스 당일)
        - [X] 총 주문 금액에서 1,000원 할인
    - [X] 증정 이벤트 (할인 전 총 주문 금액 12만원 이상)
        - [X] 25,000원 상당 샴페인 1개 증정
- [X] 12월 이벤트 배지 부여 (주문 1회 당 총 혜택 금액)
    - [X] 5,000원 이상: ⭐
    - [X] 10,000원 이상: 🎄
    - [X] 20,000원 이상: 🎅
- [ ] 이벤트 유의 사항
    - [ ] 총 주문 금액 10,000원 이상부터 이벤트 적용 가능
    - [ ] 음료만 주문시, 주문 불가
    - [ ] 메뉴 개수는 주문 1회당 최대 20개까지 가능

- [ ] 주문 내역 단계
    - [ ] 안내 문구 출력

        - [ ] 결과 출력
    - [ ] 증정 메뉴
        - [ ] 안내 문구 출력
        - [ ] 결과 출력
    - [ ] 혜택 내역
        - [ ] 안내 문구 출력
        - [ ] 결과 출력
    - [ ] 총 혜택 금액
        - [ ] 안내 문구 출력
        - [ ] 결과 출력
    - [ ] 할인 후 예상 결제 금액
        - [ ] 안내 문구 출력
        - [ ] 결과 출력
    - [ ] 12월 이벤트 배지
        - [ ] 안내 문구 출력
        - [ ] 결과 출력

#### 기본 입력 요구 사항

- [ ] 날짜 예약 단계
    - [ ] 예약일 입력
        - [ ] 예약일은 정수 타입
        - [ ] 예약일은 1 ~ 31 범위
- [ ] 주문 단계
    - [ ] 메뉴와 개수 입력
        - [ ] 메뉴와 개수는 하이픈(`-`)으로 구분
        - [ ] 메뉴와 메뉴는 쉼표(`,`)로 구분
        - [ ] 메뉴는 문자열 타입
        - [ ] 개수는 정수 타입
        - [ ] 개수는 1 ~ 20 범위

#### 기본 출력 요구 사항

- [ ] 인트로 단계
    - [ ] 안내 문구 출력

- [ ] 날짜 예약 단계
    - [ ] 안내 문구 출력
        - [ ] `12월 중 식당 예상 방문 날짜는 언제인가요? (숫자만 입력해 주세요!)`
- [ ] 주문 단계
    - [ ] 안내 문구 출력
        - [ ] `주문하실 메뉴를 메뉴와 개수를 알려 주세요. (e.g. 해산물파스타-2,레드와인-1,초코케이크-1)`
- [ ] 주문 내역 단계
    - [ ] 안내 문구 출력
        - [ ] `12월 {예약일}일에 우테코 식당에서 받을 이벤트 혜택 미리 보기!`
    - [ ] 주문 메뉴
        - [ ] 안내 문구 출력
            - [ ] `<주문 메뉴>`
        - [ ] 결과 출력
            - [ ] 출력 포맷
                - [ ] 메뉴와 개수는 공백(` `)으로 구분
                - [ ] 개수 단위 (`개`)
            - [ ] 출력 순서
                - [ ] 애피타이저
                - [ ] 메인
                - [ ] 디저트
                - [ ] 음료
                - [ ] 동일 분류 내 오름차순 출력
    - [ ] 할인 전 총 주문 금액
        - [ ] 안내 문구 출력
            - [ ] `<할인 전 총 주문 금액>`
        - [ ] 결과 출력
            - [ ] 출력 포맷
                - [ ] 금액 포맷 (`#,##0`)
                - [ ] 금액 단위 (`원`)
    - [ ] 증정 메뉴
        - [ ] 안내 문구 출력
            - [ ] `<증정 메뉴>`
        - [ ] 결과 출력
            - [ ] 출력 조건
                - [ ] 증정 메뉴 없을 경우, `없음` 출력
            - [ ] 출력 포맷
                - [ ] 메뉴와 개수는 공백(` `)으로 구분
                - [ ] 개수 단위 (`개`)
    - [ ] 혜택 내역
        - [ ] 안내 문구 출력
            - [ ] `<혜택 내역>`
        - [ ] 결과 출력
            - [ ] 출력 조건
                - [ ] 적용된 혜택 내역만 출력
                - [ ] 혜택 내역 없을 경우, `없음` 출력
            - [ ] 출력 포맷
                - [ ] 이벤트 명과 금액은 콜론(`:`)과 공백(` `)으로 구분
                - [ ] 금액은 음수로 출력
                - [ ] 금액 포맷 (`#,##0`)
                - [ ] 금액 단위 (`원`)
            - [ ] 출력 순서
                - [ ] 크리스마스 디데이 할인
                - [ ] 평일 할인 or 주말 할인
                - [ ] 특별 할인
                - [ ] 증정 이벤트
    - [ ] 총 혜택 금액
        - [ ] 안내 문구 출력
            - [ ] `<총 혜택 금액>`
        - [ ] 결과 출력
            - [ ] 출력 포맷
                - [ ] 총 혜택 금액 = 총 할인 금액 + 증정 메뉴 금액
                - [ ] 금액은 음수로 출력
                - [ ] 금액 포맷 (`#,##0`)
                - [ ] 금액 단위 (`원`)
    - [ ] 할인 후 예상 결제 금액
        - [ ] 안내 문구 출력
            - [ ] `<할인 후 예상 결제 금액>`
        - [ ] 결과 출력
            - [ ] 출력 포맷
                - [ ] 할인 후 예상 결제 금액 = 할인 전 총 주문 금액 - 총 할인 금액
                - [ ] 금액 포맷 (`#,##0`)
                - [ ] 금액 단위 (`원`)
    - [ ] 12월 이벤트 배지
        - [ ] 안내 문구 출력
            - [ ] `<12월 이벤트 배지>`
        - [ ] 결과 출력
            - [ ] 출력 조건
                - [ ] 12월 이벤트 배지 없을 경우, `없음` 출력

#### 예외 출력 요구 사항

- [ ] 날짜 예약 단계
    - [ ] 1 ~ 31 정수가 아닌 경우
        - [ ] `[ERROR] 유효하지 않은 날짜입니다. 다시 입력해 주세요.`
- [ ] 주문 단계
    - [ ] 메뉴판에 없는 메뉴를 입력한 경우
        - [ ] `[ERROR] 유효하지 않은 주문입니다. 다시 입력해 주세요.`
    - [ ] 개수가 1 이상의 정수가 아닌 경우
        - [ ] `[ERROR] 유효하지 않은 주문입니다. 다시 입력해 주세요.`
    - [ ] 개수가 20 이하의 정수가 아닌 경우
        - [ ] `[ERROR] 최대 20개까지 주문할 수 있습니다. 다시 입력해 주세요.`
    - [ ] 주문 형식이 예시와 다른 경우
        - [ ] `[ERROR] 유효하지 않은 주문입니다. 다시 입력해 주세요.`
    - [ ] 중복 메뉴를 입력한 경우
        - [ ] `[ERROR] 유효하지 않은 주문입니다. 다시 입력해 주세요.`
    - [ ] 메뉴를 음료만 입력한 경우
        - [ ] `[ERROR] 음료만 주문할 수 없습니다. 다시 입력해 주세요.`
